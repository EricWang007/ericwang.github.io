    • push()

    java.util.Stack中，加入尾部

    java.util.LinkedList中(接口Deque中规定)，加入头部。

    • add()

    加入尾部，java.util.Collection中规定

    • pop()

    java.util.Stack中，弹出尾部

    java.util.LinkedList中(接口Deque中规定)，弹出头部

    • get(int index)

    由于Deque接口中没有get方法（Deque继承自Queue），故使用LinkedList时无法用get

    而Stack继承自Vector->List，故使用Stack时可以用get

    底层实现

    Arrays.sort的底层实现原理：

    • 数据量小于等于60：使用插入排序
    • 数据量大于60：根据数据类型选择排序方式：
      • 基本类型：使用快速排序。因为基本类型相等的值都指向同一个常量池，故不需要考虑稳定性。
      • Object类型：使用归并排序。因为其具有稳定性。
